## Project Overview

This project is an end-to-end data analytics portfolio focused on business and retail data analysis.

The goal is to demonstrate professional skills in:

- SQL (MySQL)
- Python (Pandas)
- Power BI
- Data cleaning and transformation
- Business KPIs
- Data visualization and storytelling

## Technologies Used

- MySQL
- Python
- Pandas
- Power BI
- GitHub

## Project Structure

data/ -> Raw and cleaned datasets
sql/ -> SQL scripts and queries
notebooks/ -> Python & Pandas notebooks
powerbi/ -> Power BI dashboards
docs/ -> Documentation and reports
